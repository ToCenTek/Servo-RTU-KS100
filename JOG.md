手动寸动(JOG)设置：

| 序号  | 参数    | 值      | 地址       | 功能       | 说明                                                                                                         |
| --- | ----- | ------ | -------- | -------- | ---------------------------------------------------------------------------------------------------------- |
| 1   | PA-4  | 1      | `0x0004` | 控制方式选择   | 0：位置控制方式；<br/>1：速度控制方式；<br/>2：转矩控制方式；<br/>3：位置速度混合控制方式；<br/>4：位置转矩混合控制方式；<br/>5：速度转矩混合控制方式；<br/>6：编码器调零模式。 |
| 2   | PA-22 | 3      | `0x0016` | 速度指令来源   | 0=模拟量 <br/>1=内部速度 <br/>2=模拟+内部 <br/>3=JOG <br/>4=键盘 <br/>5=IO                                              |
| 3   | PA-53 | 1      | `0x0035` | 伺服强制使能   | 0=使能信号由 DI 输入的 SON 控制 <br/>1=软件强制使能                                                                        |
| 4   | PA-60 | 1      | `0x003C` |          | 0：软复位无效；<br/>1：软复位有效，系统将重新启动。<br/>或手动重新上电生效                                                                |
| 5   | Jr-   |        |          | JOG运行    | 在面板选择Jr-进入JOG运行模式                                                                                          |
| 6   |       |        |          |          | 按住 ^ 键正转 或 v 键反转，默认100转/分钟                                                                                 |
| 7   | PA-21 | 0-6000 | `0x0035` | JOG 运行速度 | 默认100                                                                                                      |
