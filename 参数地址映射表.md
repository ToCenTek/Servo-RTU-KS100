# KS100 伺服参数地址映射表

## 地址编码规则

| 参数组  | 面板标记 | 等效标记 | 寄存器地址公式      | 地址范围            |
| ---- | ---- | ---- | ------------ | --------------- |
| PA 组 | FA-* | PA-* | `0x0000 + N` | 0x0000 ~ 0x007F |
| P3 组 | FC-* | P3-* | `0x0100 + N` | 0x0100 ~ 0x012F |
| P4 组 | FD-* | P4-* | `0x0200 + N` | 0x0200 ~ 0x022F |

> **PA 暂存地址** = 保存地址 + `0x0080`（写入后立即生效，断电不保存）

## Modbus 常用命令速查

### 通讯参数推荐设置

- **PA-71 (0x0047)** = 1 (站号)
- **PA-72 (0x0048)** = 96 (9600bps)
- **PA-73 (0x0049)** = 0 (8N2)
- **FA-4 (0x0004)** = 3 (内部位置模式)
- **FA-14 (0x000E)** = 3 (内部位置输入)
- **FA-53 (0x0035)** = 1 (软件强制使能)

### 内部位置控制流程

1. 写 `FC-31 (0x011F)` bit0 = 0 → 停止当前运动
2. 写目标参数：`FD-2~FD-4` (圈数/脉冲/速度)
3. 写 `FC-31` bit0 = 1 → 触发运动

### HOLD 停止

- 写 `FC-31 (0x011F)` bit0 = 0, bit1 = 1 → HOLD 停止
  - 需 FC-39 (0x0127) = 27 (HOLD)

### Servo Enable

- 写 `FA-53 (0x0035)` = 1 → 使能
- 写 `FA-53 (0x0035)` = 0 → 关闭

### Set Home

- 写 `FC-34 (0x0122)` = 1 → 设当前位置为零点（编码器坐标清零）
